QuranCode supports any tafseer in any language.

  QuranCode requires one *.htm tafseer file per sura:
  (e.g. 001000.htm for surat Al-Fatiha.)

  QuranCode requires one *.htm tafseer file per aya:
  (e.g. 001001.htm for Bism Allah Ar-Rahmaan Ar-Raheem.)

  If you wish to add a tafseer in your own language, then
  please format it as one *.htm file per sura and per aya,
  and zip it and email it to us at heliwave@yahoo.com.
  We will in sha Allah validate it and add it to QuranCode.

  We reserve the right to correct/reject any tafseer at anytime.
